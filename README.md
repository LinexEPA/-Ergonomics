ErgoLift 是一個基於瀏覽器的輕量級人因工程分析工具，專為職業安全評估與工業工程（IE）應用設計。透過電腦視覺輔助與生物力學模型，協助評估搬運作業時的腰椎（L5/S1）負荷風險。

✨ 功能特色

🛡️ 隱私優先 (Privacy First)：採用純前端技術 (HTML5/JS)，所有影片分析與計算皆在使用者的瀏覽器端完成，影片不會上傳至任何伺服器。
📐 影片角度測量：內建簡易的關鍵點標記工具（Kinovea 風格），可上傳工作影片並測量軀幹彎曲角度。
📊 即時力學計算：基於 2D 平面生物力學模型，即時計算 L5/S1 椎間盤壓力。
📉 可視化圖表：提供壓力與角度的趨勢圖，並標示 NIOSH 建議的安全界限 (3400N)。
🚀 無需安裝：單一 HTML 檔案，下載即用，或透過網頁直接操作。

🚀 如何使用
上傳影片：點擊「上傳影片」按鈕，選擇您拍攝的作業影片（支援 MP4/MOV）。

標記關鍵點：
播放影片至動作最大（最彎腰）的瞬間並暫停。
依序點擊畫面上的 髖關節 (Hip) 與 肩膀 (Shoulder)。
系統將自動計算軀幹彎曲角度。
調整參數：在右側面板輸入實際的「物體重量」、「作業者體重」與「負重距離」。
查看報告：系統會即時顯示 L5/S1 壓力值與風險等級（低/中/高）。

🛠️ 技術細節
核心架構：Vanilla JavaScript (ES6+)
UI 框架：Tailwind CSS (透過 CDN)
圖表繪製：Chart.js
圖示庫：Lucide Icons

⚠️ 免責聲明

本工具僅供人因工程初步評估與學術參考使用。計算結果基於簡化的生物力學模型，不可替代專業醫療診斷或詳細的職業醫學鑑定。如有嚴重的肌肉骨骼不適，請諮詢專業醫師或物理治療師。
